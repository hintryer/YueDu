# YueDu
📚 自用的一些「阅读」小说书源，一部分是网上搜集/他人推荐，一部分是自己写的书源规则。  

## 温馨提示：
> 注意！因为**精力有限**加上我现在基本上只看收藏的完本小说，所以 **`本项目书源较少，维护也不积极`**，因此**请勿过度依赖**本项目，想要更多书源的话可以去 [源仓库](https://www.yckceo.com/yuedu/shuyuan) 看看。
